1. Getting Started

Before you start working with this theme, make sure that you have WordPress already installed on your server.

1.1 Installation

To install a theme, you must first upload the theme files and then activate the theme in WordPress admin panel. Use one of this 2 options:

  • WordPress Upload: Navigate to Appearance > Add New Themes > Upload. Go to browse, and select the zipped theme folder. Hit “Install Now” and the theme will be uploaded and installed.
  • FTP Upload: Using your FTP program, upload the non-zipped theme folder into the /wp-content/themes/ folder on your server.

Once the theme is uploaded, go to Appearance > Themes and activate your new theme.

1.2 Settings

Go to Settings > Reading and set Front page to display your latest posts. While being here you can set how many post should be presented on each blog page.

Go to Users > Your Profile and uncheck "Show Toolbar when viewing site". It's important, because with WordPress toolbar enabled, sometimes there is unwanted padding under the navigation bar.

I recommend to change Permalinks in Settings > Permalinks to some other option than default. Typically I use Post name.

1.3 Polish characters

If you are building site in Polish language, or need some additional Latin Extended characters go to Editor, chose header.php from the file list and change this fragment of code:

<link href='http://fonts.googleapis.com/css?family=Gentium+Basic:400,700,400italic' rel='stylesheet' type='text/css'>
<link href='http://fonts.googleapis.com/css?family=Open+Sans:700,400' rel='stylesheet' type='text/css'>

to this:

<link href='http://fonts.googleapis.com/css?family=Gentium+Basic:400,700,400italic&subset=latin,latin-ext' rel='stylesheet' type='text/css'>
<link href='http://fonts.googleapis.com/css?family=Open+Sans:400,700&subset=latin,latin-ext' rel='stylesheet' type='text/css'>

2. Intro section

Intro is a fullscreen section, which people visiting your website will see first. To create this section, go to Intro > Add New. Please create only 1 intro, because theme is developed in a way, that it could show only 1 section like that.

You can enter your welcome text, and upload background photo now. Make sure that image is in landscape view, and have proportions similar to 16:9. Photo used in demo has a resolution of 1920px x 1200px.

If your image is light, you can choose darker color of text and button.

If you just need a simple blog, without intro section - just don't create one, it's that simple.

3. Header

3.1 Site title

You can enter title of your site, and tagline in Settings > General. If you don't need a tagline, leave the field empty, and your title will magically center itself in the header.

3.2 Logo

Instead of using title as a text, there is also an option to upload custom logo. If that's what you prefer, go to Apperance > Customize. Expand Logo section and upload your image. It is recommended that your logo height won't be higher than 40px. Remember to click save button.

3.2 Navigation

Before you can create menu, remember that first you need to publish some content. Create some pages, publish some posts. If you have some content already published, go to Apperance > Menus and create your new menu there.

Tip: To create Home in menu use Links panel, enter your website url (http://www.yourwebsite.com), and enter Home as Link Text.

Tip: If you want to show blog categories under the dropdown menu (Categories > Category 1, Category 2 etc.) - use Links panel, enter # as url, and Categories as Link Text. Add selected categories under just created menu item (move them slightly to the right). Remove # from firstly created item.

3. Blog

3.1 Image size

It is recommended, that blog post images, and featured image will have a width of at least 820px. 820px is the width of the site, so the images will occupy whole available space, and align nicely with other elements.

3.2 Lightbox

This theme has integrated lightbox feature. If you have large images and want to show them in bigger size when clicked:

  • click "Add media" in post editing screen, upload your image
  • set "Link to" -> Media File, and "Size" -> Full size

If you don't need lightbox feature, choose "Link to" -> none, when adding new images.

3.3 Categories

Categories, assigned to posts, will show under the excerpt on the blog page. If you don't want to display them, uncheck "Show post category" in post editing screen.

3.4 Excerpt

To show post summary on main blog page, please use Excerpt field in post editing screen. If you don't see that field, go to Screen options (at the top), and check "Excerpt".

5. Footer widgets

Daily comes with 3 widget ready footer areas. You can insert widgets in Apperance > Widgets, using Footer 1, 2 and 3 panels.
